UnitedAV

An original, Apache-2.0, FFmpeg/LGPL audio-video plugin for Unity — web backends (WebGL + WebGPU) over the browser's own video stack.

WebGPU showcase WebGPU
A ring of video screens with an orbiting camera — the Unity Showcase, ported to a standalone WebGPU 3D scene.
WebGPU player
Full-window playback. WebCodecs + a WGSL shader apply the clip's signalled colorspace (exact colour with software decode; browser-conversion fallback otherwise).
WebGL player
Full-window playback via texImage2D(video) into a WebGL2 texture.
Mic & speaker test
WebAudio tone to the speaker + getUserMedia mic capture with live level meters. Click to grant the mic.
WebGPU verifier
Decode → GPUTexture → readback, checked against a 2D-canvas reference + ffmpeg oracle.
Native plugin: CPU + hardware decode (VAAPI / VideoToolbox / D3D11 / NVDEC) with zero-copy GPU present (GL / D3D11 / Metal / Vulkan). The web pages use the browser's own decode + WebGL/WebGPU upload, behind the same C# surface. Built & deployed by Nix + GitHub Actions.
